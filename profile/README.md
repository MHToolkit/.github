# MH Toolkit

Cross-platform toolkit for classic Monster Hunter emulation, overlays, save sync, and multiplayer on macOS and Android.

## Repositories

- [`mh-toolkit`](https://github.com/MHToolkit/mh-toolkit) — planning, specs, governance, and integration roadmap.
- [`mh-overlay`](https://github.com/MHToolkit/mh-overlay) — overlay tooling for health and damage display.
- [`nemessix`](https://github.com/MHToolkit/nemessix) — independent 3DS emulator product line used by the first phase.

Website: <https://mhtoolkit.github.io>
