# MH Toolkit

MH Toolkit is an organization-level toolkit ecosystem for classic Monster Hunter emulation on macOS and Android.

The hub repository describes the vision, architecture, repository map, onboarding path, CI/CD orchestration and coordinated releases. Heavy implementation lives in focused child repositories.

## Repositories

- [`mh-toolkit`](https://github.com/MHToolkit/mh-toolkit) — organization hub: vision, specs, governance, issue routing, integration gates and coordinated releases.
- [`mh-overlay`](https://github.com/MHToolkit/mh-overlay) — overlay implementation for health and damage display.
- [`nemessix`](https://github.com/MHToolkit/nemessix) — independent 3DS emulator product line used by the first phase.
- [`MHToolkit.github.io`](https://github.com/MHToolkit/MHToolkit.github.io) — public organization website.

Website: <https://mhtoolkit.github.io>
